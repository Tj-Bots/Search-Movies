# ***חיפוש סרטים***
בוט לחיפוש סרטים בטלגרם.

# **מה יש בבוט?**
<b>⌨️ פקודות למשתמשים:</b>
<coust>/start - התחל בוט.
/stats - סטטיסטיקות הבוט.
/id - קבלת מזהה משתמש/צ'אט.
/info - קבלת מידע על משתמש כמו: פרופיל, שם, יוזר, איידי וכו'...
/tts - תמלל הודעת טקסט לשמע.
/font - משנה טקסט באנגלית לטקסט יפה.
/share - שיתוף טקסט.
/paste - העלאת קובץ טקסט/הודעת טקסט לאתר Pastebin.</coust>
